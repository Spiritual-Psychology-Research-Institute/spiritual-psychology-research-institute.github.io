## 페이지 구성

- **Hero** — “내면의 지도 / 사이間” 메시지, 숲 실루엣 비주얼 + 한자 「間」 카드 + 김윤희 박사 인용
- **01 소개** — 인사말, 풀쿼트, 학력·경력 그리드 (심리학 학사 / 상담심리학 석사 / 영성(심리)학 박사 외)
- **02 상담 안내** — 개인상담 · 심리검사 카드 + 4단계 상담 절차
- **03 오시는 길** — 상동역 7번 출구 위치를 표현한 SVG 지도
- **Closing / Contact** — 전화·이메일 CTA
- **Footer** — 4컬럼

## 로컬에서 보기

`index.html` 한 파일로 구성된 정적 사이트입니다. 빌드 단계가 없습니다.

```sh

open index.html

# 또는 간단한 정적 서버
python3 -m http.server 8000
```
